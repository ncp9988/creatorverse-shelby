## Installation

1. **Clone the repository:**
```bash

 git clone https://github.com/ncp9988/creatorverse-shelby.git
   cd creatorverse-shelby 
   ```

2. **Install dependencies:**
```bash
npm install
```


4. **Run the app:**
```bash
npm start
```
